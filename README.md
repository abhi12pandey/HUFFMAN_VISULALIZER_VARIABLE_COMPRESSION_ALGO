# 🌳 Huffman Coding Visualizer

An interactive web-based visualization tool for understanding **Huffman Coding (Variable-Length Encoding)** with real-time tree construction, encoding, and decoding.

---

## 🚀 Features

- 🌳 Dynamic Huffman Tree Visualization (SVG-based)
- ⚡ Step-by-step Animation of Tree Construction
- 🔐 Encode Text using Huffman Coding
- 🔍 Decode Visualization (bit-by-bit traversal)
- 📊 Compression Statistics (Original vs Encoded size)
- 🖱️ Zoom & Drag Tree Interaction
- 📂 File Input Support (.txt)

---

## 🧠 Concept

Huffman Coding is a **lossless compression algorithm** where:
- Frequent characters → shorter codes  
- Rare characters → longer codes  

This project demonstrates:
- Tree construction using Min Heap
- Prefix-free binary encoding
- Efficient data compression

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Frontend)
- SVG for Tree Visualization
- C++ (Reference Implementation)
